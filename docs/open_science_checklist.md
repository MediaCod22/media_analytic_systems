# Open Science checklist

## Репозиторий

- [ ] README описывает цель, объект, предмет и исследовательские вопросы.
- [ ] Указана структура репозитория.
- [ ] Указана версия данных, соответствующая поданной статье.
- [ ] Добавлена лицензия.
- [ ] Добавлен CITATION.cff.

## Методология

- [ ] Есть PRISMA-протокол.
- [ ] Есть поисковая стратегия по каждой базе.
- [ ] Есть критерии включения и исключения.
- [ ] Есть кодировочная книга.
- [ ] Есть описание расчета DSS-связи.
- [ ] Есть раздел ограничений и смещений.

## Данные

- [ ] Есть агрегированные производные таблицы.
- [ ] Есть таблица PRISMA flow counts.
- [ ] Есть объяснение, какие исходные данные нельзя публиковать и почему.
- [ ] Есть метаданные набора данных.

## Воспроизводимость

- [ ] Есть скрипт проверки таблиц.
- [ ] Проверяется сумма 112 + 95 + 120 = 327.
- [ ] Проверяется разрыв 35 / 8 ≈ 4,4.
- [ ] Проверяется наличие пяти режимов аналитической работы.

## Для подачи статьи

- [ ] В статье указана ссылка на репозиторий.
- [ ] В статье указана дата обращения к репозиторию.
- [ ] В статье указано, какие материалы доступны открыто.
- [ ] При использовании ИИ в подготовке текста добавлено disclosure statement.
