# GitHub release notes draft

## v1.0-open-science-companion

Релиз содержит открытый исследовательский пакет к статье «Режимы аналитической работы с медиаданными: от измерений к поддержке принятия решений».

### Добавлено

- PRISMA-протокол отбора публикаций.
- Поисковая стратегия для Google Scholar, eLibrary/РИНЦ и CNKI.
- Кодировочная книга исследования.
- Агрегированные таблицы по выборке, DSS-связи и типологии режимов.
- Open Science checklist.
- Метаданные набора данных и CITATION.cff.
- Скрипт валидации производных таблиц.

### Основные числа

- 804 первично найденных публикаций.
- 327 публикаций в итоговой аналитической выборке.
- 112 публикаций Google Scholar.
- 95 публикаций РИНЦ.
- 120 публикаций CNKI.
- 477 публикаций в расширенном российском массиве.
- DSS-связь: Google Scholar — 13%, РИНЦ — 8%, CNKI — 35%.
- Разрыв Россия/Китай по DSS-связи — 4,4 раза.
