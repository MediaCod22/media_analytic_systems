# Системы медиааналитики: Наукометрическое исследование

## Обзор проекта

Данный репозиторий содержит материалы комплексного наукометрического исследования систем медиааналитики и их интеграции с системами поддержки принятия решений (DSS — Decision Support Systems). Исследование охватывает период 2019-2025 годов и включает анализ публикаций из трёх наукометрических баз данных.

## Ключевые результаты

### Сравнительная таблица по базам данных

| Показатель | Google Scholar | eLibrary/РИНЦ | CNKI (Китай) |
|------------|----------------|---------------|--------------|
| Выборка публикаций | 112 | 95 | 120 |
| Упоминание систем | 16% | 33% | 45% |
| **Связь с DSS** | **13%** | **8%** | **35%** |

### Главный вывод

**Разрыв между Россией и Китаем по интеграции медиааналитики с DSS составляет 4,4 раза (8% vs 35%)**

Российские системы технически развиты, но не институционализированы для принятия решений.

## Методология

- **Период исследования:** 2019-2025 гг.
- **Базы данных:** Google Scholar, eLibrary/РИНЦ, CNKI
- **Выборка:** 327 публикаций (наукометрия) + 477 публикаций (обзор РИНЦ)
- **Критерии анализа:** упоминание систем, связь с DSS, тематика

## Ключевые выводы

1. **Российские системы развиты технологически** — не уступают зарубежным по функционалу
2. **Низкая DSS-интеграция** — только 8% публикаций связывают медиааналитику с принятием решений
3. **Китайский опыт** — 舆情监测 встроена в систему государственного управления (35% DSS-связь)
4. **Потенциал Талисмана** — академическая разработка с наивысшим DSS-потенциалом (33%)
