# Системы медиааналитики: Наукометрическое исследование

## Обзор проекта

Данный репозиторий содержит материалы комплексного наукометрического исследования систем медиааналитики и их интеграции с системами поддержки принятия решений (DSS). Исследование охватывает период 2019-2025 годов и включает анализ более 800 публикаций из трёх наукометрических баз данных.

## Ключевые выводы

| Показатель | Google Scholar | eLibrary/РИНЦ | CNKI (Китай) |
|------------|----------------|---------------|--------------|
| Выборка публикаций | 112 | 95 | 120 |
| Упоминание систем | 16% | 33% | 45% |
| **Связь с DSS** | **13%** | **8%** | **35%** |

**Главный вывод:** Разрыв между Россией и Китаем по интеграции медиааналитики с DSS составляет **4,4 раза** (8% vs 35%)

## Исследованные системы

### Российские системы
- **Медиалогия** — 1253 публикации, индекс Хирша: 25, DSS-связь: 6%
- **Brand Analytics** — ~340 публикаций, DSS-связь: 13%
- **СКАН-Интерфакс** — ~95 публикаций, DSS-связь: 8%
- **Крибрум** — ~180 публикаций, DSS-связь: 7%
- **Талисман (ИСП РАН)** — ~45 публикаций, DSS-связь: 33%

### Зарубежные системы
- **Palantir** — 2100+ публикаций, индекс Хирша: 89, DSS-связь: 65%
- **Factiva** — ~890 публикаций
- **LexisNexis** — ~720 публикаций
- **Meltwater** — ~340 публикаций

## Методология

- **Период исследования:** 2019-2025 гг.
- **Базы данных:** Google Scholar, eLibrary/РИНЦ, CNKI
- **Выборка:** 327 публикаций (наукометрия) + 477 публикаций (обзор РИНЦ)
- **Критерии анализа:** упоминание систем, связь с DSS, тематика

## Использование

Материалы могут быть использованы для:
- Написания научных статей по медиакоммуникациям
- Разработки курсов по медиааналитике
- Проектирования DSS с интегрированной медиааналитикой
