## 5. Open-source инструменты медиааналитики (GitHub)

| Инструмент | GitHub звёзды | Форки | Назначение | Язык | Применение в DSS |
|------------|---------------|-------|------------|------|------------------|
| **Sherlock** | 73 830 ⭐ | 8 769 | Поиск аккаунтов по username в 400+ соцсетях | Python | Соцмедиа-разведка |
| **awesome-osint** | 25 343 ⭐ | 3 471 | Кураторский список OSINT-инструментов | - | Ресурс для исследователей |
| **SpiderFoot** | 17 007 ⭐ | 2 853 | Автоматизация OSINT, визуализация связей | Python | Threat intelligence |
| **Twint** | 16 347 ⭐ | 2 776 | Twitter scraping без API | Python | Анализ соцмедиа |

### Описание инструментов:

**1. Sherlock (73 830 ⭐)**
- Назначение: поиск социальных аккаунтов по имени пользователя
- Охват: 400+ социальных сетей
- Применение: OSINT-разведка, выявление цифрового следа
- Особенность: не требует API, работает напрямую
- DSS-потенциал: выявление фиктивных аккаунтов, анализ репутации

**2. SpiderFoot (17 007 ⭐)**
- Назначение: автоматизация OSINT для threat intelligence
- Интеграции: 200+ данных из открытых источников
- Визуализация: графы связей между сущностями
- DSS-потенциал: поддержка принятия решений в области безопасности

**3. Twint (16 347 ⭐)**
- Назначение: сбор данных из Twitter без API
- Возможности: скрейпинг твитов, подписчиков, хештегов
- Ограничение: Twitter/X изменил политику доступа
- DSS-потенциал: анализ общественного мнения

**4. theHarvester (15 849 ⭐)**
- Назначение: сбор информации о доменах и организациях
- Данные: email, subdomains, IP, имена сотрудников
- Источники: поисковые системы, PGP, LinkedIn
- DSS-потенциал: корпоративная разведка

**5. Obsei (1 382 ⭐)**
- Назначение: low-code AI-автоматизация для social listening
- Возможности: анализ тональности, алертинг, brand monitoring
- Интеграции: Twitter, Reddit, Facebook, AppStore
- DSS-потенциал: интеграция в корпоративные workflows

## 7. Выводы по системам

### 7.1. Российские системы
- **Технологически развиты** — не уступают зарубежным аналогам по функционалу
- **Низкая DSS-интеграция** — используются преимущественно для мониторинга, не для принятия решений
- **Потенциал Талисмана** — академическая разработка с высоким DSS-потенциалом (33%)

### 7.2. Китайские системы
- **Высокая DSS-интеграция** — 舆情监测 встроена в систему государственного управления
- **Собственные AI-модели** — независимость от Western технологий
- **Институционализация** — медиааналитика как элемент Governance

### 7.3. Open-source инструменты
- **Высокая популярность** — Sherlock имеет 73 000+ звёзд
- **Ограниченная интеграция** — требуют технической экспертизы
- **DSS-потенциал** — SpiderFoot и Obsei наиболее пригодны для интеграции

### 7.4. Ключевой разрыв
Разрыв между Россией (8%) и Китаем (35%) по DSS-интеграции объясняется не технологиями, а **институциональными факторами** и **методологической неразработанностью**.
