# Поисковая стратегия

## Общая логика

Поиск строился вокруг пересечения трех смысловых блоков:

1. медиаданные и медиааналитика;
2. аналитические системы и платформы;
3. поддержка принятия решений, прогнозирование и risk / reputation intelligence.

## Рекомендации для финальной версии репозитория

1. Для каждого запроса добавить дату выполнения.
2. Для каждого запроса добавить количество найденных результатов.
3. Зафиксировать язык интерфейса и региональные настройки поиска.
4. Для Google Scholar указать, как контролировались дубликаты и нерелевантные результаты.
5. Для CNKI описать процедуру перевода и нормализации китайских терминов.
