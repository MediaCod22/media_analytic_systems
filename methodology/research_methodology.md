# Методология исследования

## 1. Цели и задачи исследования

### 1.1. Цель
Провести комплексный наукометрический анализ публикаций по тематике медиааналитики для выявления степени интеграции медиаданных в процессы принятия решений.

### 1.2. Задачи
1. Систематизировать публикации по медиааналитике в трёх наукометрических базах
2. Оценить частоту упоминания конкретных систем медиааналитики
3. Выявить связь публикаций с системами поддержки принятия решений (DSS)
4. Провести сравнительный анализ между Россией, Китаем и мировым уровнем
5. Определить ключевые научные школы и авторов

## 2. Источники данных

## 4. Критерии отбора публикаций

### 4.1. Временной период
2019-2025 годы (7 лет с учётом неполного 2025 года)

### 4.2. Типы публикаций
- Статьи в научных журналах
- Материалы конференций
- Монографии и главы книг
- Препринты (для Google Scholar)

### 4.3. Критерии исключения
- Дубликаты
- Нерелевантные темы (технические спецификации, реклама)
- Публикации без аннотации

## 5. Методы анализа

### 5.1. Количественный анализ
- Подсчёт количества публикаций по годам
- Оценка доли статей в ядре РИНЦ
- Индекс Хирша для систем и авторов

### 5.2. Качественный анализ
- Тематическая классификация (6 направлений)
- Оценка связи с DSS по ключевым словам
- Выявление научных школ

### 5.3. Критерий DSS-связи

Публикация считалась связанной с DSS при наличии:
- Упоминания "decision support", "DSS", "СППР"
- Ключевых слов "принятие решений", "decision making"
- Контекста использования медиаданных для решений

## 6. Ограничения исследования

1. **Доступ к проприетарным системам** — не проводился анализ внутри систем Медиалогия, СКАН и др.

2. **Языковой барьер** — китайские источники анализировались по аннотациям и ключевым словам

3. **Динамическая индексация** — данные Google Scholar и CNKI приблизительны

4. **Неполный 2025 год** — данные за 2025 год частичны (по ноябрь)

## 7. Статистическая обработка

- Расчёт долей и процентов
- Сравнительный анализ между базами
- Выявление трендов по годам
- Корреляционный анализ (где применимо)

## 8. Воспроизводимость

Все исходные данные сохранены в репозитории:
- `/data/` — исходные данные
- `/tables/` — сводные таблицы
- `/results/` — результаты анализа
